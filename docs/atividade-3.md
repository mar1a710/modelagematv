# Atividade: Linha do Tempo do Projeto (Visão Sênior)

> **Abordagem Técnica:** Foco estrito em entregáveis essenciais (MVP) e prazos realistas para um ciclo ágil de desenvolvimento de software.

## 📊 Matriz de Planejamento

| ETAPA | O QUE FAREMOS? | PRAZO |
| :--- | :--- | :--- |
| **Análise** | Elicitação de requisitos críticos, alinhamento com stakeholders e definição do escopo final. | 2 Semanas |
| **Projeto** | Modelagem do banco de dados (DER), arquitetura de APIs e prototipagem de alta fidelidade (UI/UX). | 3 Semanas |
| **Implementação** | Desenvolvimento iterativo do código (Frontend e Backend) e revisões de código (*Code Review*). | 6 Semanas |
| **Teste** | Automação de testes (unitários/integração) e homologação final com o cliente (UAT). | 3 Semanas |
| **Manutenção** | Configuração do pipeline de CI/CD, monitoramento de performance e correções pós-deploy. | Contínuo |

---
*💡 **Nota de Engenharia:** Os prazos foram estimados utilizando IA como ferramenta de apoio para análise histórica de tarefas similares, mitigando riscos de gargalos na fase de implementação.*
